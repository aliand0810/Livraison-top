📖 Comment configurer Google Cloud ?
  1. Allez sur console.cloud.google.com
  2. Créez un projet ou sélectionnez-en un
  3. Activez l'API Google Drive API
  4. Allez dans Identifiants → Créer des identifiants → ID client OAuth
  5. Type : Application Web
  6. Dans « URI de redirection autorisées », collez l'URI ci-dessus
  7. Copiez le Client ID et collez-le ci-dessous
  8. Dans Écran de consentement OAuth, ajoutez votre email en tant qu'utilisateur test
